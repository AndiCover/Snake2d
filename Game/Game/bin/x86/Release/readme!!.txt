1. Installieren
Entpacke die Zip datei.

2. Starten
Starte die Game.exe um das Spiel zu spielen.

3. Systemanforderungen:
DirectX 9.0c oder höher
XNA Framework 4.0 oder höher
.Net Framework 3.5 oder höher
Bildschirmauflösung von 1024x768 oder höher

WICHTIG!!
Wenn Sie das XNA Framework nicht installiert haben, dann installieren sie es vor dem start!

Sie können es hier downloaden:
http://www.microsoft.com/download/en/details.aspx?id=20914
